# 系统演示视频方案

> 目标：自动化生成系统操作演示视频，演示已开发好的系统，告诉用户怎么点、怎么使用。

## 技术方案

**Playwright（自动化操作） + TTS（语音旁白） + Remotion（视频合成）**

### 整体流程

1. **Playwright 脚本** — 定义操作流程（点击、输入、滚动等），驱动浏览器自动执行并录制视频片段
2. **Markdown 文案** — 编写每一步的解说词（复用现有文案流程）
3. **TTS 语音合成** — 将解说词生成语音旁白（复用现有 TTS 能力）
4. **Remotion 合成** — 将视频片段 + 语音旁白 + 注释动画合成为最终视频

### 关键技术点

- **Playwright codegen**：在浏览器上操作时自动生成脚本代码，降低编写门槛
- **Playwright 视频录制**：初始化 context 时设置 `record_video_dir` 即可自动录制
- **人性化操作模拟**：随机打字延迟、鼠标高亮、点击波纹、操作停顿
- **场景化拆分**：教程拆分为多个场景，每个场景 = 浏览器操作 + 旁白文本，方便维护
- **Remotion**：用 React 编写视频，可加步骤标注、转场动画、鼠标高亮等

### 核心优势

- **可维护**：系统 UI 变更后，改脚本即可一键重新生成视频
- **可版本管理**：脚本 + 文案均为代码，可 Git 管理
- **可 CI/CD**：可集成到流水线，自动化生成最新演示视频
