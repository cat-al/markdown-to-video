# 手机可读性优化 — 全局放大 + 色系对比度增强

**日期**：2026-04-27
**状态**：已批准

## 问题

1. **字体太小**：当前 HTML 幻灯片以 1920×1080 固定视口设计，在手机上播放视频或浏览器预览时，画面被 ~5 倍缩放，18px 标注缩至 ~4px，基本不可读。
2. **色系区分度低**：背景 `#FAF3E9`（纸浆米白）与主色 `#A8703F`（陶土棕）、强调色 `#C4923C`（暖金棕）都处于暖黄-棕色调区间（Hue 30°-40°），明度差仅 ~35%，小屏幕上对比度不足。

两个场景都受影响：
- 最终视频在手机上播放（抖音/B站/微信等竖屏播放横屏视频）
- 直接在手机浏览器里打开 `presentation.html` 预览

## 方案：全局放大 + 色系对比度增强（方案 A）

核心思路：视口尺寸 1920×1080 不变（录屏流程需要），在创作规范层面提高最小字号基线，调整色系对比度。

### 不改什么

- 视口尺寸 1920×1080 不变
- 背景色 `#FAF3E9` 不变
- 纸浆纹理 `paper-texture-bg.png` 不变
- TimelineEngine、Step 编排系统、录屏流程均不变
- 品牌调性（暖色系为主）不变

## 字号规范

### 最小字号基线

| 元素类型 | 当前 | 优化后 | 手机 ~5x 缩放后 |
|---------|------|--------|----------------|
| 页面标题 | 42px | **64px+** | ~13px ✓ |
| 节点内文字 | 18px | **32px+** | ~6px 可辨认 |
| 内联标注（SVG `<text>` / HTML） | 13-14px | **28px+** | ~6px 可辨认 |
| 引导文案（顶部/底部一句话） | 18px | **40px+** | ~8px ✓ |
| 数字锚点 | 72px 起步 | **96px 起步，重要数字 160px+** | ~19-32px ✓ |
| 辅助说明文字 | 14px | **28px+** | ~6px 可辨认 |
| 序号/编号（圆圈内） | 22px | **36px+** | ~7px ✓ |
| 要点标题 | 20px | **36px+** | ~7px ✓ |

### 配套调整

- **节点尺寸放大**：节点从 160×60 → 220×80 左右，给更大字号留空间
- **连线加粗**：stroke-width 从 2px → 3-4px
- **描边加粗**：卡片/容器描边从 1px → 2-3px
- **信息密度降低**：每页内容更少，严格执行"放不下就拆页"原则

## 色系规范

### 色值对照

| 角色 | 当前色值 | 优化色值 | 变化 |
|------|---------|---------|------|
| 背景 | `#FAF3E9` | `#FAF3E9` | 不变 |
| 主色（陶土棕） | `#A8703F` | **`#7A4F2A`** | 加深，明度差 35% → 52% |
| 强调（暖金棕） | `#C4923C` | **`#B8802E`** | 提饱和度 |
| 辅助（橄榄绿） | `#5C614D` | **`#4A6741`** | 偏绿更明显，冷暖对比 |
| 文字（深棕） | `#2a2218` | **`#1a1408`** | 加深，接近纯黑的深棕 |
| 文字弱化 | `rgba(42,34,24,0.58)` | **`rgba(26,20,8,0.58)`** | 基色加深 |
| 警告（赤陶红） | `#a33020` | **`#8B2515`** | 加深 |

### CSS 变量更新

```css
:root {
  --color-bg: #FAF3E9;           /* 不变 */
  --color-bg-strong: #EDE5D5;    /* 略深，与 bg 拉开 */
  --color-text: #1a1408;         /* 加深 */
  --color-text-muted: rgba(26, 20, 8, 0.58);
  --color-accent: #7A4F2A;       /* 陶土棕 — 加深 */
  --color-accent-warm: #B8802E;  /* 暖金棕 — 提饱和 */
  --color-danger: #8B2515;       /* 赤陶红 — 加深 */
  --color-focus: #4A6741;        /* 橄榄绿 — 偏绿 */
}
```

### 色系设计原则

- **背景不动，前景加深**：拉大明度差，保持品牌基调
- **冷暖对比**：辅助色从灰绿偏移到明确的绿，与棕色系形成冷暖区分
- **色彩语义不变**：陶土棕=正常/主要、暖金棕=强调/过渡、赤陶红=错误/失控、橄榄绿=辅助/聚焦
